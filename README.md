# UbnerithUML

**UbnerithUML** est une application desktop de modélisation UML permettant de concevoir et gérer l’ensemble des diagrammes UML courants au sein d’un même document.

---

## Description

L’application fournit un éditeur graphique dédié à la création de diagrammes UML, avec une approche orientée document : un projet peut contenir plusieurs diagrammes de types différents, tous éditables et persistants.

---

## Diagrammes UML supportés

- Diagramme de classes
- Diagramme de séquence
- Diagramme d’activités
- Diagramme d’états-transitions
- Diagramme de cas d’utilisation

---

## Import / Export

- **Export PNG**  
  Export du diagramme courant en image.

- **Export JSON**  
  Sauvegarde complète du document UML via une fenêtre système.

- **Import JSON**  
  Chargement d’un document UML existant via une fenêtre système.

---

## Technologie

- Application desktop basée sur **Tauri**
- Interface développée en **React + TypeScript**
